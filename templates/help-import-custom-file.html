Importing Reviews Using a Custom File

Import reviews from CSV or Excel files for maximum flexibility

Overview

If you have reviews from other platforms, manual collections, or custom formats, you can import them using a CSV or Excel file. This method gives you complete control over which reviews to import.

Supported File Formats

  • CSV (Comma-Separated Values) - Most common, works with Excel, Google Sheets
  • Excel (.xlsx) - Microsoft Excel format
  • Excel (.xls) - Older Excel format

Required Columns

Your file must include these columns (column names are case-insensitive):

Column Name Required Description Example
shopify_product_id ‚úÖ Yes Your Shopify product ID 10045740417338
reviewer_name ‚úÖ Yes Customer's name John D.
rating ‚úÖ Yes Star rating (1-5) 5
body or review_text ‚úÖ Yes Review content Great product! Highly recommend.
title ‚ùå No Review title Amazing quality
images or photo_urls ‚ùå No Comma-separated image URLs https://example.com/img1.jpg,https://example.com/img2.jpg
review_date ‚ùå No Date (YYYY-MM-DD or any format) 2025-01-15
reviewer_country ‚ùå No Country code (US, UK, etc.) US
verified_purchase ‚ùå No true/false or yes/no true

Step-by-Step Guide

Step 1: Prepare Your File

Create a CSV or Excel file with your reviews. Here's a sample format:

shopify_product_id,reviewer_name,rating,body,title,images,review_date,reviewer_country,verified_purchase 10045740417338,John D.,5,"Great product! Works perfectly.","Amazing quality","https://example.com/img1.jpg,https://example.com/img2.jpg",2025-01-15,US,true 10045740417338,Sarah M.,4,"Good value for money.","Nice product","",2025-01-10,UK,true 10045740417338,Mike T.,5,"Highly recommend!","Excellent","https://example.com/img3.jpg",2025-01-08,CA,false

Step 2: Access Import Tool

  1. Go to your Sakura Reviews dashboard
  2. Click "Import Reviews"
  3. Select "Import from File"
  4. Click "Choose File" and select your CSV/Excel file

Step 3: Map Columns

If your column names don't match exactly, you'll see a column mapping screen where you can match your columns to our required fields.

Step 4: Preview & Import

  1. Review the preview of your reviews
  2. Check for any errors or warnings
  3. Select which reviews to import (or import all)
  4. Click "Import Selected Reviews"

Common Issues & Fixes

‚ùå "Column not found" Error

Problem: Your file is missing a required column.

Solution: Add the missing column to your file. Column names are case-insensitive, so "Reviewer Name" = "reviewer_name".

‚ùå "Invalid rating" Error

Problem: Rating values are not between 1-5.

Solution: Ensure all ratings are whole numbers between 1 and 5. Convert percentages (e.g., 100% = 5 stars, 80% = 4 stars).

‚ùå "Invalid product ID" Error

Problem: Shopify product ID doesn't exist in your store.

Solution: Verify the product ID is correct. Find it in your Shopify admin: Products ‚Üí [Product] ‚Üí Check URL.

‚ùå "Images not loading"

Problem: Image URLs are invalid or inaccessible.

Solution: Ensure image URLs are publicly accessible (not behind authentication). Use direct image links (ending in .jpg, .png, etc.).

Best Practices

  • Clean Your Data: Remove duplicates and invalid entries before importing
  • Use Direct Image URLs: Images must be publicly accessible
  • Format Dates Consistently: Use YYYY-MM-DD format for best results
  • Test with Small Files First: Import 5-10 reviews first to verify format
  • Keep Backups: Save your original file before importing
üí° Pro Tip: You can export reviews from other platforms (Judge.me, Yotpo, etc.) and import them directly using this method!

File Size Limits

  • Maximum file size: 10 MB
  • Maximum rows: 10,000 reviews per file
  • For larger imports: Split into multiple files
